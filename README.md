Installation:
    Make sure you have Node.js and npm installed. Then, install dependencies:
        npm install
    If using a backend, navigate to the backend folder and install dependencies:
        cd backend
        npm install    

Technologies Used:
    Backend: Node.js, Express, MongoDB
